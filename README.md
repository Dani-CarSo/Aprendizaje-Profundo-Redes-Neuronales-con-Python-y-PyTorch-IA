# Aprendizaje-Profundo-Redes-Neuronales-con-Python-y-PyTorch-IA
Proyecto de IA.
Las redes neuronales constan de un conjunto de "neuronas", que son valores que parten de los datos de entrada, se multiplican por pesos, se suman y luego se pasan por una función de activación para producir nuevos valores. Este proceso se repite en todas las "capas" que tenga la red neuronal para producir una salida.
